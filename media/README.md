# Media Folder

Place your video files here for automatic random playback!

## Supported Formats
- `.mp4` (Recommended)
- `.webm`
- `.mov`

## How It Works
1. Add video files to this folder
2. The website will automatically detect and load them
3. Videos play randomly
4. When one ends, another random video plays

## Recommendations

### File Size
- Keep videos under 50MB for best performance
- Compress videos using tools like HandBrake

### Resolution
- 1080p (1920x1080) recommended
- 720p (1280x720) for faster loading
- 4K works but may be slow on some devices

### Format Settings
For MP4 files, use:
- Codec: H.264
- Bitrate: 2-5 Mbps
- Audio: AAC, 128-192 kbps

## Example Structure
```
media/
├── video1.mp4
├── video2.mp4
├── anime-scene.mp4
└── gaming-montage.mp4
```

## Testing
1. Add videos to this folder
2. Open profile.html
3. Videos should auto-play randomly
4. Check browser console for any errors

## Fallback
If no videos are found, the website will:
- Show animated gradient background
- Allow manual URL input via settings

## Tips
- Use short loops (10-30 seconds) for best effect
- Mute audio in videos (website handles music separately)
- Test on mobile devices for performance
